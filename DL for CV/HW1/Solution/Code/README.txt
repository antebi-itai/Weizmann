Instructions
------------
  1. Open your Google Drive in the browser.
  2. Create a directory called "DL4CV" in your Google Drive.
  3. In "DL4CV", create a sub-directory called "hw1".
  4. Upload the files listed above to "DL4CV/hw1".
  5. Double-click on "hw1.ipynb" to open it in Google Colab.
  6. Follow the instructions in the notebook and solve the coding assignment.
  7. Solve the written assignment and upload your solution to "DL4CV/hw1" as "hw1-sol.pdf".
  8. Run the last cell in the notebook to create a submission zip file and download it.
  9. Upload your submission zip to the assignment's submission box (a Dropbox linked; sent in Piazza).
